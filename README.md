# S&P 500 Volatility Analysis: Pre, During & Post COVID-19

Analyzed S&P 500 market behavior across three economic periods using FRED data. Computed daily returns, annualized volatility, and mean  returns to examine the effects of pandemic shocks, fiscal stimulus,  and monetary policy intervention.

**Tools:** Python, pandas, NumPy, matplotlib  
**Data Source:** Federal Reserve Economic Data (FRED)
